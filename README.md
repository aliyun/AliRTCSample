# 音视频通信RTC示例程序

音视频通信RTC 提供了一组示例程序，展示如何集成阿里云RTC SDK 以便于用户开发自己的视频通话应用。

## 平台支持

示例程序支持 Android、iOS、Windows、Mac 平台。

## 演示功能

- 加入 / 退出频道
- 视频开启 / 关闭
- 视频订阅 / 取消订阅
- 摄像头切换
- 静音 / 取消静音
- 扬声器 / 耳机切换
